# Dachak Gold Shop  
Online Gold & Jewelry Store | WordPress + WooCommerce
Live Website:  
https://dachakgold.com/
Project Overview  
Dachak Gold Shop is an online gold and jewelry store developed using WordPress and WooCommerce.  
I handled the full setup, including UI/UX design, theme customization, feature implementation, and performance optimization.
The project was built with a focus on smooth user experience, scalable structure, and easy product management.
User-Facing Features
- Fully responsive and mobile-friendly layout  
- Structured product categories  
- Customized product page layout  
- High-quality image gallery  
- Search and filtering based on product attributes
Development & Technical Features
- WordPress and WooCommerce setup and configuration  
- Custom theme modifications and template overrides  
- Product attributes configuration (weight, karat, model, etc.)  
- Installation and setup of essential store plugins  
- Performance optimization (caching, image optimization, CDN-ready setup)  
- Basic security hardening and access protection
Repository Structure  
This repository does not include the full WordPress installation or database.  
It is intended to present documentation, and project details as a portfolio sample.
